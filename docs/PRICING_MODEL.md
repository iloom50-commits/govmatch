## 1. 요금제 개요

"단계별 가치 잠금 해제" 기반의 4단계 구조. 비회원부터 전문 컨설턴트까지 사용자 유형별로 필요한 기능을 차등 제공.

**과금 원칙**: 비용이 발생하는 AI 기능(Gemini API 호출)을 유료화하고, 서버 비용이 거의 없는 검색/알림은 무료 제공.

## 3. 단계별 상세 설명

### 3.1 비회원 (가입 전)

- **이용 가능**: 공고 검색 + 공고 상세 페이지 이동
- **이용 불가**: 맞춤 알림, AI 상담, 저장, 프로필 관리
- **설계 의도**: 서비스 가치를 먼저 확인 → "이 공고 나한테 맞나?" 궁금증 유발 → 가입 유도

### 3.2 FREE (무료 회원)

- **대상**: 처음 가입하는 사용자
- **핵심 가치**: 맞춤 공고 알림 + 공고AI 상담 1회 체험
- **공고AI 상담 1회**: 특정 공고를 선택하여 "우리 회사가 지원 가능한지" AI가 정밀 분석
- **설계 의도**:
  - 1회 상담으로 AI 가치를 체험 → "다른 공고도 물어보고 싶다" → LITE 전환 동기
  - 검색/알림은 무료 → 서비스에 머무르게 하여 전환 기회 극대화

### 3.3 LITE (월 2,900원)

- **대상**: 정기적으로 지원금을 탐색하는 소상공인/사업자
- **핵심 가치**: 공고AI 상담 무제한
- **FREE 대비 추가**: 공고AI 상담 무제한 (특정 공고별 자격요건 정밀 상담)
- **설계 의도**:
  - "3천원 미만" 심리적 진입 장벽 최소화 (커피 한 잔)
  - 공고별 상담이 핵심 니즈 → 이것만으로 충분한 가치 제공
  - 자유AI/컨설턴트는 미포함 → PRO와 명확한 차별화

### 3.4 PRO (월 19,900원)

- **대상**: 전문 컨설턴트, 세무사, 경영지도사 등 고객사를 대리하는 전문가
- **핵심 가치**: 자유AI 상담 + AI 컨설턴트 무제한 + 전문가 도구
- **LITE 대비 추가**:
  - 자유AI 상담 무제한: 공고를 안 골라도 "우리 회사에 맞는 거 찾아줘" 가능
  - AI 컨설턴트 무제한: AI가 대화로 고객 조건 수집 → 가상 프로필 생성 → 매칭
  - (Coming Soon) 고객사 프로필 다건 관리
  - (Coming Soon) 상담 이력 엑셀 다운로드
  - (Coming Soon) 종합 리포트 (고객별 지원 가능 공고 분석)
- **설계 의도**:
  - 컨설턴트에게 월 2만원은 고객 1건 수임료 대비 푼돈 → 지불 의향 높음
  - 가격 자체가 "전문가용"이라는 포지셔닝 역할
  - LITE 대비 7배 → 일반 사용자는 LITE에, 전문가는 PRO에 자연 분류

## 7. 비용 구조 분석

### 서버 비용이 발생하는 기능 (Gemini API 호출)
- 공고AI 상담 → API 호출 1회/대화
- 자유AI 상담 → DB 검색 + API 호출/대화
- AI 컨설턴트 → API 다단계 호출/대화

### 서버 비용이 거의 없는 기능
- 공고 검색/상세 → DB 조회만
- 맞춤 알림 → 스케줄러 기반
- 매칭 결과 보기 → DB 조회만

## 9. 횟수 기반 과금을 채택하지 않은 이유

초기에 AI 상담 횟수 기반 과금을 검토했으나 다음과 같은 문제로 기각:

1. **불공정한 차감**: 창을 열기만 해도 1회 차감 → 사용자 불만
2. **사용 심리 위축**: "횟수가 줄어든다"는 인식이 서비스 이용 자체를 억제
3. **이탈 리스크**: 월 200회 중 10회만 사용 시 → 다음달 구독 취소 가능성 높음
4. **가치 인식 불일치**: "남은 횟수"에 집중하게 되어 서비스 가치보다 소비량에 주목

→ **기능 등급제**로 전환하여 "이 등급에선 이 기능을 무제한 사용 가능"이라는 명확한 가치 전달.
