콘텐츠 노출 및 클릭
웹마스터도구에서 제공하는 콘텐츠 노출 및 클릭 정보는 본인이 등록한 웹사이트가 네이버 검색결과에 얼마나 노출되고 클릭 되었는지를 보여주는 성과 리포트입니다.

제공 기준
검색결과에서의 노출/클릭 산정 영역
네이버 검색결과에서 웹마스터도구의 노출/클릭에 산정되는 영역은 웹 검색과 연관된 영역에 대해서만 제공합니다. VIEW 및 블로그 검색영역을 비롯하여, 기업/학교/명소/게임등과 같이 컨텐츠 정보 영역에 포함된 지표, 검색 광고영역은 노출 및 클릭 산정에 포함되지 않습니다.

기준일
노출/클릭 정보는 "업데이트 기준일"로 부터 "최근 90일" 까지 데이터만 제공하며 업데이트 기준일은 1주 전의 검색 데이터를 기준으로 산정됩니다. 만약 최근 7일치에 대한 클릭/노출 데이터를 조회한다면 현재일로부터 2주 전 ~ 3주 전 네이버 검색창에 노출된 데이터를 보는 셈입니다.

제공되는 정보
클릭수 : 네이버 검색결과에서 내 사이트의 콘텐츠가 얼마나 클릭 되었는지 횟수
노출수 : 네이버 검색결과에서 내 사이트의 콘텐츠가 얼마나 노출 되었는지 횟수
CTR : 클릭된 횟수를 노출된 횟수로 나눈 값 (클릭수 / 노출수 * 100)
예를 들어 'A'라는 키워드에 대하여 사용자가 네이버 검색결과에 입력시 여러분의 사이트 콘텐츠가 100 번 노출되고 그중에서 3번이 클릭되었다면 CTR은 3%가 됩니다. 그러므로 CTR이 높은 콘텐츠는 검색사용자가 원하는 정보를 가지고 있다는 의미입니다.

데이터 보관
기준일 부터 90일 이전까지의 데이터를 보관하고 있으며 소유확인된 사이트를 삭제하는 경우 해당 사이트의 노출/클릭 데이터도 같이 삭제 됩니다. 단, 사이트의 소유자 변경시에는 삭제되지 않습니다.

노출/클릭 요약
기간별 (1일/7일/30일/60일/90일) 콘텐츠의 노출 및 클릭정보를 제공하며 아래와 같이 3가지 지표로 분류합니다.
ctr리포트

최근 총 클릭
웹 검색 노출 영역에서 기간별로 여러분의 웹사이트를 클릭한 지표이며, 과거의 동일 구간 대비 증감 정보를 제공합니다.

예를 들어 최근 7일 동안 클릭지표가 1만이라고 가정해 본다면, 검색 사용자가 특정 키워드를 입력후 노출되는 검색 결과에서 여러분의 사이트가 제공하는 콘텐츠를 클릭하여 실제 사이트 방문까지 이어지는 일련의 과정이 1만번 발생하였다는 의미입니다. 또한, 지난 기간 대비 증감표시의 경우는 최근 7일 기준 조회이므로 최근 14일전~최근 7일까지의 클릭 수를 의미합니다.

예)
최근 업데이트 : 2019.10.22
조회 기간 : 최근 7일
조회 장비 : PC + Mobile
클릭 지표 : 1만
설명
10.16일 ~ 10.22일(7일) 동안, 검색 사용자가 PC와 Mobile 환경으로 검색한 결과에서 내 사이트가 제공하는 콘텐츠를 클릭 하고 방문한 횟수 1만건
지난 기간 대비 증감는 10.09 ~ 10.15 기간의 클릭 지표 합과 10.16 ~ 10.22 기간의 클릭 지표 합으로 증감율 계산하여 표시
최근 총 노출
웹 검색 노출 영역에서 기간별로 여러분의 웹사이트가 노출된 지표이며, 과거의 동일 구간 대비 증감 정보를 제공합니다.

예를 들어 최근 7일 동안 노출지표가 1만이라고 가정해 본다면, 검색 사용자가 특정 키워드를 입력후 노출되는 검색 결과에서 여러분의 사이트의 콘텐츠가 포함된 횟수를 의미합니다. 노출 순위는 검색결과의 첫 페이지일 수도 있고 마지막 페이지 일 수도 있습니다. 이 지표는 여러분의 콘텐츠가 검색 로봇 수집 후 색인 과정을 거쳐서 사용자의 키워드에 매칭되어 검색 노출까지 이어지는 일련의 과정을 통과하였다는 지표로 활용할 수 있습니다.

예)
최근 업데이트 : 2019.10.22
조회 기간 : 최근 30일
조회 장비 : Mobile
노출 지표 : 2백
설명
09.23일 ~ 10.22일(30일) 동안, 검색 사용자가 Mobile 환경에서 특정 키워드를 검색한 결과에 내 사이트가 제공하는 콘텐츠 2백건 노출
지난 기간 대비 증감는 08.24 ~ 09.22 기간의 노출 지표 합과 09.23 ~ 10.22 기간의 노출 지표 합으로 증감율 계산하여 표시
검색 키워드 및 웹문서 TOP 30
기간별 (1일/7일/30일/60일/90일) 로 클릭 수가 가장 높은 검색 키워드 및 URL에 대하여 상위 30건의 정보를 제공합니다. 이를 통해 기간별로 어떠한 콘텐츠가 사용자의 선택을 받고 있는지 유추할 수 있습니다.
