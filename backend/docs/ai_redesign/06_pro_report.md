## 6. 생성 파이프라인

```python
def generate_pro_report(session_id, db_conn):
    # 1. 세션 전체 데이터
    session = fetch_session(session_id)
    client_profile = fetch_client_profile(session.client_profile_id)
    messages = session.messages
    matched = session.matched_snapshot

    # 2. 대화에서 expert_insights 누적
    accumulated_insights = {}
    for m in messages:
        if m.role == "assistant" and "expert_insights" in m.meta:
            accumulated_insights[m.meta.announcement_id] = m.meta.expert_insights

    # 3. 공고 상세 조회
    ann_details = {aid: fetch_announcement(aid) for aid in accumulated_insights.keys()}

    # 4. Gemini 요약 호출 (Exec Summary + 로드맵)
    ai_summary = gemini_generate_summary({
        "client": client_profile,
        "matched": matched,
        "insights": accumulated_insights,
    })

    # 5. 컨설턴트 브랜딩 정보
    branding = fetch_consultant_branding(session.business_number)

    # 6. 템플릿 렌더
    html = render_template("pro_full.j2", {
        "client_profile": client_profile,
        "matched": matched,
        "insights": accumulated_insights,
        "ann_details": ann_details,
        "ai_summary": ai_summary,
        "branding": branding,
    })

    # 7. PDF 생성
    pdf_bytes = html_to_pdf_with_fonts(html)

    # 8. 저장
    report_id = save_to_client_reports(
        client_profile_id=session.client_profile_id,
        owner_bn=session.business_number,
        content=html,
        pdf=pdf_bytes,
    )

    return {"report_id": report_id, "html": html, "pdf_url": f"/api/pro/reports/{report_id}/pdf"}
```

## 7. API

```
POST /api/pro/reports/generate
  body: {session_id}
  response: {report_id, html, pdf_url}

GET /api/pro/reports
  query: ?client_id=...  (옵션)
  response: [{report_id, client_name, date, summary}, ...]

GET /api/pro/reports/{id}
GET /api/pro/reports/{id}/pdf
POST /api/pro/reports/{id}/edit-section  ← 컨설턴트 편집
POST /api/pro/email/send  ← 고객에게 이메일 발송
```

기존 엔드포인트 대부분 존재. Schema 재정비만 필요.

## 8. 컨설턴트 편집 기능

- 섹션별로 `[편집]` 버튼
- 편집 모드 진입 → 텍스트 수정 → 저장 → 버전 기록
- 최종 확정 후 PDF 재생성

---

## 9. 고객 발송

- 이메일 (SMTP)
- 카카오톡 (옵션, 컨설턴트 카카오 연동)
- 보고서 링크 공유 (30일 만료 token)

---

## 10. 리스크

- PDF 생성 시간 (5~15초) → 비동기 처리 + 진행 표시
- 보고서 분량 과다 → 섹션별 "핵심만 보기" 토글
- 컨설턴트 편집 충돌 → 편집 lock / 버전 관리
