# 지원금매칭 요금제 모델

> 최종 확정일: 2026-03-18
> 상태: 설계 완료 (구현 예정)

---

## 1. 요금제 개요

기능 등급제 기반의 3단계 구독 모델. 비용이 발생하는 AI 기능(Gemini API 호출)을 기준으로 유료화하고, 서버 비용이 거의 없는 직접 입력 매칭은 모든 등급에서 무제한 제공.

---

## 2. 요금제 비교표

| 구분 | Free (0원) | Basic (4,900원/월) | Business (19,900원/월) |
|------|-----------|-------------------|----------------------|
| **공고 매칭 (직접 입력)** | 무제한 | 무제한 | 무제한 |
| **매칭 결과 보기** | 전체 | 전체 | 전체 |
| **AI 상담 (자유 대화)** | 일 1회 | 무제한 | 무제한 |
| **AI 컨설턴트 (대화형)** | X | X | 무제한 |

| **PDF 리포트** | X | X | O |
| **맞춤 알림** | O | O | O |

---

## 3. 등급별 상세 설명

### 3.1 Free (무료)

- **대상**: 처음 방문한 사용자, 가볍게 둘러보는 사용자
- **핵심 가치**: 직접 입력 매칭으로 서비스 가치를 먼저 체험
- **제한 사항**:
  - AI 상담(자유 대화): 하루 1회로 제한
  - AI 컨설턴트 대화형: 이용 불가
  - 신청 가이드 / PDF 리포트: 이용 불가
- **설계 의도**:
  - 직접 입력 매칭은 Gemini API 호출 없음 → 서버 비용 거의 0
  - 무제한 매칭으로 서비스 유용성을 충분히 경험하게 하여 자연스러운 업그레이드 유도
  - AI 상담 일 1회 제공으로 AI 기능의 가치도 맛볼 수 있음

### 3.2 Basic (월 4,900원)

- **대상**: 정기적으로 지원금을 탐색하는 소상공인/스타트업
- **핵심 가치**: AI 상담 무제한으로 궁금한 점을 자유롭게 질문
- **Free 대비 추가 기능**:
  - AI 상담(자유 대화) 무제한
  - 신청 가이드 제공
- **설계 의도**:
  - AI 상담이 가장 많이 사용되는 기능 → 무제한 제공으로 확실한 가치 전달
  - 월 4,900원은 "커피 한 잔" 수준의 심리적 부담이 낮은 가격

### 3.3 Business (월 19,900원)

- **대상**: 지원금을 본격적으로 활용하는 기업, 다수 공고에 지원하는 사용자
- **핵심 가치**: AI 컨설턴트 + PDF 리포트로 전문적인 지원금 컨설팅
- **Basic 대비 추가 기능**:
  - AI 컨설턴트 대화형 무제한
  - PDF 리포트 제공
- **설계 의도**:
  - AI 컨설턴트 대화형은 다단계 Gemini API 호출 → 가장 비용이 높은 기능
  - PDF 리포트로 실질적인 비즈니스 자료 제공
  - Basic의 약 4배 가격이지만, 컨설턴트 기능의 가치가 충분히 높음

## 4. 추천 보상 시스템

| 항목 | 내용 |
|------|------|
| **추천인 보상** | Basic 30일 무료 체험 |
| **피추천인 보상** | Basic 30일 무료 체험 |
| **적용 조건** | 피추천인이 가입 완료 시 양쪽 모두 자동 적용 |

### 설계 의도
- Free 사용자가 추천만으로 Basic을 체험 → 결제 전환율 향상
- 추천인/피추천인 모두 동일한 보상 → 공정성 확보
- "30일 Basic 체험"은 구독 의사결정에 충분한 기간

## 5. 비용 구조 분석

### 서버 비용이 발생하는 기능 (Gemini API 호출)
- AI 상담 (자유 대화) → API 호출 1회/대화
- AI 컨설턴트 (대화형) → API 다단계 호출/대화
- PDF 리포트 생성 → API 호출 + PDF 렌더링

### 서버 비용이 거의 없는 기능
- 공고 매칭 (직접 입력) → DB 조회만
- 매칭 결과 보기 → DB 조회만
- 맞춤 알림 → 스케줄러 기반

### 과금 원칙
> **비용이 발생하는 기능을 유료화하고, 비용이 없는 기능은 무제한 제공**

이 원칙에 따라 직접 입력 매칭은 모든 등급에서 무제한이며, AI 기능은 등급에 따라 차등 제공.

---

## 6. 횟수 기반 과금을 채택하지 않은 이유

초기에 AI 상담 횟수 기반 과금을 검토했으나 다음과 같은 문제로 기각:

1. **불공정한 차감**: 창을 열기만 해도 1회 차감 → 사용자 불만
2. **사용 심리 위축**: "횟수가 줄어든다"는 인식이 서비스 이용 자체를 억제
3. **이탈 리스크**: 월 200회 중 10회만 사용 시 → 다음달 구독 취소 가능성 높음
4. **가치 인식 불일치**: "남은 횟수"에 집중하게 되어 서비스 가치보다 소비량에 주목

→ **기능 등급제**로 전환하여 "이 등급에선 이 기능을 무제한 사용 가능"이라는 명확한 가치 전달.
